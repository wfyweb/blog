杂谈——我是前端开发工程师

2017,6,10 生活 Hits

好久没有写点什么了，但是今天想写点什么。就有一点感觉我腰好疼啊，好累啊，可为什么没有感到充实那，没有写代码的快感,因为自己写的太少了，可能没有做自己想要做的事。略略略，副标题已过，下面说下这段时间的工作。

刚开始工作还挺轻松的，仿个网页，看看文档，了解后台，看看功能。不对看文档，不轻松，我发现我语文能力，阅读理解能力有问题，看文档好累，这方面要加强，作为一个程序员，看文档，哼哼，要加强。有空还玩玩游戏，都堕落了。然后就转到文投公司，做文化产业网。这个项目直观感觉高大上，不过有点懵。给小伙伴们科普下，政府项目，提升文化产业发展，展现当地文化面貌，推动文化企业，经济进步。为文化企业一站式服务。项目资金申报，新媒体宣传，网站建设，vi设计，技术开发，法律咨询，会计托管。哎呦脑壳疼，以上只是个人观点。然后开展工作，前期就是大量开会，头脑风暴，思想碰撞，每天回去，感觉身体被掏空，然而不高兴的是，敲代码时间少了。一致于今天，遇到字体滚动都懵了，不会写了，还是基础没学好啊。有空把这个dome写十遍，我是前端开发工程师啊，好丢人啊。略略略，写十遍，写好贴上让大家看。对就是，我的定位前端开发工程师，工程师，工程师。我不能忘了我热爱的事情，记得有句话，人生最幸运的是在年轻的时候遇到自己热爱的事情，并且这个事情还能赚钱。乔老爷子说的。这段时间的工作，打乱了自己学习路线，具体总结，开展，node, angular vue php考虑考虑，最近看hadoop真6，不过有linux java c python 基础，我的天啊，都全栈工程师了，任重道远。每天根据下班时间，制定学习时间，最可怕的是不坚持。我热爱的事情，我会做到。

最后一段，发个广告，许昌文化产业网，值得你信赖，因为有我。没了，今天6月8号，高考结束了。记得我的dome........和我的定位....

对了现在眼前是ps..公司需要，好尴尬啊！
